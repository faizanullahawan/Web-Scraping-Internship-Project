# Web Scraping Internship Project

## Project Overview
This project demonstrates web scraping using Python and BeautifulSoup. The goal is to extract structured data from a public website, convert it into a dataset, and perform basic analysis.

## Tools & Technologies
- Python
- BeautifulSoup
- Requests
- Pandas
- Matplotlib / Seaborn

## Objectives
- Extract data from a public website
- Parse HTML structure
- Store data in structured format (CSV)
- Perform basic data analysis
- Visualize insights from dataset

## Dataset Information
The dataset includes:
- Title / Name
- Price (if available)
- Rating (if available)
- Availability or Category

## Project Workflow
1. Send HTTP request to website
2. Parse HTML using BeautifulSoup
3. Extract required fields
4. Store data in list/dictionary
5. Convert into Pandas DataFrame
6. Save as CSV file
7. Perform data analysis and visualization
